# sass-learn


## Basics 1 (project1)
``` bash
# 命令行指令
$ compass create <project>

# 编译Sass
$ sass <sass file> <css file> --style
  -- 4种输出风格
     -- nested  
     -- expanded
     -- compact
     -- compressed

# 监视效果
$ sass --watch <sass file>:<css file>

$ compass compile

# 监视文件夹
$ sass --watch <sass folder>:<css folder>

$ compass watch  --force(强制编译)  -(输出风格)

```
